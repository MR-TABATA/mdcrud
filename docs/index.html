Markdown ビューア

Tauri + TypeScript で作られた、シンプルなデスクトップ Markdown ビューア。開いて、読むだけ。

A minimal, fast and safe Markdown viewer built with Tauri. Just open and read.

ダウンロード / Download GitHub で見る

特徴 / Features

必要なものだけ。余計なものは何もない。

軽量・高速

Tauri ネイティブで起動も描画も一瞬。Electron より小さく、軽い。

Native Tauri build — tiny binary, instant startup.

安全に表示

DOMPurify でサニタイズ + CSP。信頼できない Markdown を開いても安心。

HTML sanitized with DOMPurify & CSP — safe by default.

サッと開く・自動更新

ダブルクリックやドラッグ&ドロップで開き、保存すれば自動で再描画。最近のファイルからも一発。

Open by double-click or drag & drop, auto-reload on save, plus a recent-files list.

ダークテーマ

目にやさしい VS Code 風の配色。長文も快適に読める。

Eye-friendly VS Code-style dark theme.

表・コード・図

テーブル、コード、引用、ローカル画像に加え、Mermaid 図もそのまま描画。

Tables, code, quotes, local images, and Mermaid diagrams.

クロスプラットフォーム

macOS / Windows / Linux 向けに配布。動作確認は macOS で実施済み。

Builds for macOS, Windows and Linux. Verified on macOS.

ショートカット / Shortcuts

手を止めずに操作。

⌘/CtrlO
ファイルを開く / Open
⌘/CtrlR
再読み込み / Reload

はじめる / Get Started

プレビルド版をダウンロード / Download a prebuilt binary

最新版をダウンロード / Latest Release

macOS / Windows / Linux 向けバイナリを配布。Apple Developer 公証は未対応のため、初回起動時は OS の警告を回避する必要があります（手順）。

または、ソースからビルド / Or build from source

# リポジトリを取得 / Clone
$ git clone https://github.com/MR-TABATA/mdcrud.git
$ cd mdcrud

# 依存をインストール / Install
$ npm install

# 開発モードで起動 / Run in dev
$ npm run tauri dev

# 配布用にビルド / Build for release
$ npm run tauri build
